These are John Cowan's C-sources for Joy1, the new version of Joy. The sources have been compiled on various Unix and non-Unix systems apparently without any problems. (If you do not have the BDW garbage collected version yet, go back to the main page to fetch the current joy.tar.gz .)

Use the files here (or just the joy1src.tar.gz at the end) 

          IF you only have limited discspace
          OR you are not running unix,
          OR you wish to update just the C-sources:
  • globals.h
  • main.c
  • scan.c
  • utils.c
  • interp.c
  • make.nogc
    ( Unix) Use the command line: make -f make.nogc
    (Non-Unix) Use the C compiler/linker as appropriate
  • online-man.txt online manual for (raw) Joy1
  • joy1src.tar.gz gzipped tar file of all the above files
